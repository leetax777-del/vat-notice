02준비서류

어떤 서류를 준비해야 하나요

아래 자료는 별도로 제출하지 않으셔도 됩니다
  • 전자로 발행된 전자세금계산서 또는 계산서
  • 법인신용카드(개인사업자는 홈택스 등록 사업용카드) 사용내역
  • 신용카드·제로페이·현금영수증 매출자료

매출관련자료 — 공통

  • 종이로 발급한 수기세금계산서 또는 계산서
  • 신용카드·제로페이·현금영수증 외 별도 매출(지역화폐 등)이 있는 경우 해당 매출내역서
  • 현금영수증 미발행 계좌이체 매출이 있는 경우, 해당 기간 매출 합계액

매출관련자료 — 업종별

수출업 전자상거래업 콘텐츠제작업 음식점업 프랜차이즈업 약국·병원 부동산임대업
  • 수출업 — 수출내역을 입증할 수 있는 서류 (인보이스, 구매확인서, 수출신고필증 등)
  • 전자상거래업 — 플랫폼(네이버, 쿠팡, 이니시스, 카카오 등) 매출내역서
  • 콘텐츠제작업 — 영상플랫폼(유튜브, 인스타, 메타, 틱톡 등) 매출내역서
  • 음식점업 — 중개플랫폼(배민, 요기요, 쿠팡이츠, 땡겨요 등) 매출내역서
  • 프랜차이즈업 — 본사 지원금·쿠폰 등이 있는 경우 사이트 다운로드 월별매출내역서
  • 약국·병원 — 요양급여·의료급여 매출내역서, 플랫폼·결제대행사 이용 시 해당 매출내역서
  • 부동산임대업 — 부동산 임대 변동내역
참고 새롭게 거래하게 된 플랫폼이 있다면 놓치지 말고 담당자에게 전달해 주세요. 외화로 매출을 수령하는 경우 외화통장 거래내역과 입금확인서 등 입증서류도 함께 준비해 주세요.
대표적 전자상거래 플랫폼별 매출자료 다운로드 방법 예시

네이버 스마트스토어

  1. 정산관리 메뉴로 이동
  2. 부가세 신고내역 선택
  3. 기간을 2026.01.01 ~ 06.30으로 설정 후 다운로드
  4. 항목별 정산내역 → 혜택 정산 → 세금신고기준일 2026.01.01 ~ 06.30으로 검색 후 다운로드

네이버페이

  1. 네이버페이센터 로그인 후 2차 인증 진행
  2. 상단 메뉴에서 정산관리 → 부가세신고 선택
  3. 조회기간 설정 후 검색, 월별내역 다운으로 엑셀파일 저장
  4. 필요 시 해당 화면 캡처 또는 다운로드한 파일을 담당자에게 전달

쿠팡윙

  1. 쿠팡윙 로그인 후 정산 메뉴로 이동
  2. 부가세 신고내역 선택
  3. 기간을 2026.01.01 ~ 06.30으로 설정 후 검색
  4. 상세다운로드 후 담당자에게 전달

토스페이먼츠

  1. 토스페이먼츠 상점관리자 로그인 후 내 상점 선택
  2. 결제 조회 → 부가세 신고 자료 메뉴 선택
  3. 조회 기간(분기·반기·연간) 설정 후 조회
  4. 엑셀 다운로드 후 담당자에게 전달

다날

  1. 다날 파트너센터 로그인
  2. 정산/세금 → 부가세 신고 자료 조회 메뉴 선택
  3. 조회 기간 설정 후 검색
  4. 엑셀 다운로드 후 담당자에게 전달

매입관련자료

  • 종이로 받은 수기세금계산서 또는 계산서
  • 홈택스 미등록 카드 지출내역 (개인사업자)
  • 대표자·직원 개인카드를 업무용으로 사용한 경우, 카드사명·전체번호·지출내역
전달 방법 홈택스 미등록 카드 지출내역은 엑셀파일로 전달해 주세요. 거래처의 사업자등록번호가 반드시 포함되어야 합니다.
확인 필수 개인카드 지출내역 전달 시 카드번호는 마스킹(****) 없이 전체 번호로 보내주세요.
